F# ScriptでPythonの__name__を真似する

    Pythonでは以下のように記述することで、最初に起動されたのかimportされたのかを判断することができます。

    test1.py
    if __name__ == '__main__':
    print("main")
else:
    print("not main")
    test2.py
    import test1
    実行結果
    $ python test1.py
main

$ python test2.py
not main

    これをF# Scriptで真似してみました。

    Test1.fsx
    if System.Diagnostics.StackTrace().GetFrame(0).GetMethod().Name = "main@" then
    printfn "main"
else
    printfn "not main"
    Test2.fsx
    #load "Test1.fsx"
    実行結果
    > fsi Test1.fsx
main

> fsi Test2.fsx
main

> fsc Test1.fsx
Microsoft (R) F# Compiler version 11.0.50727.1
Copyright (c) Microsoft Corporation. All Rights Reserved.

> fsc Test2.fsx
Microsoft (R) F# Compiler version 11.0.50727.1
Copyright (c) Microsoft Corporation. All Rights Reserved.

Test2.fsx(1,18): warning FS0988: Main module of program is empty: nothing will happen when it is run

> Test1
main

> Test2
not main

    コンパイルしないとうまく動かないのが微妙です。私の用途ではF# Scriptをコンパイルして使っているので事足りますが、もっとうまい方法があるかもしれません。
